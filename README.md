# K-Means Clustering & Elbow Method

Реализация алгоритма кластеризации K-means "с нуля" на языке Python с процедурой автоматического поиска оптимального числа кластеров.

## Основные возможности
* **K-means:** итеративный алгоритм группировки точек по евклидову расстоянию.
* **Метод «локтя» (Elbow Method):** автоматический расчет оптимального количества $k$ путем поиска максимального удаления от прямой на графике SSE.
* **Визуализация:** генерация изображений процесса кластеризации с помощью библиотеки PIL.
* **Генерация данных:** создание случайных кластеров с нормальным распределением.

## Требования
Для работы программы необходимы библиотеки:
* `numpy`
* `Pillow`
* `matplotlib`
